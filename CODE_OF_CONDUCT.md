# Code of Conduct  
FactCheck BD Project

## Purpose  
This Code of Conduct aims to ensure that everyone contributing to FactCheck BD—whether through discussions, documentation, design, or development—participates in a respectful, safe, and inclusive environment. The project is civic-oriented and prioritizes public benefit, ethical practices, and responsible technology use.

## Our Standards  
All contributors are expected to:

- Treat others with respect and professionalism  
- Welcome differing viewpoints and constructive feedback  
- Avoid harassment, discrimination, or personal attacks  
- Maintain a focus on improving the project and serving users  
- Follow principles of ethical, transparent, and safe technology development  

The following behaviors are not acceptable:

- Harassment or intimidation of any kind  
- Discriminatory language or actions  
- Sharing personal information (doxxing)  
- Politically targeted attacks or manipulation  
- Misuse of the project to harm individuals or communities  

## Scope  
This Code of Conduct applies to all project communication channels, including GitHub issues, pull requests, documentation contributions, discussions, and future community spaces.

## Reporting Issues  
If you witness or experience behavior that violates this Code of Conduct, please report it through:

- GitHub issues (with a clear description)
- Direct message to project maintainers (when applicable)

All reports will be handled discreetly and respectfully.

## Enforcement  
Project maintainers may take action including warnings, issue blocking, or removal from participation if the Code of Conduct is violated. Decisions will be made fairly and consistently.

## Commitment  
FactCheck BD is committed to building technology that strengthens information integrity while protecting the rights, safety, and dignity of all contributors and users.

Thank you for helping maintain a healthy and constructive environment.
